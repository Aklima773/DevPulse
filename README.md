             2.PostgreSQL
             3.Next

 API endpoint list: 1.post/api/auth/signup

                    3.post/api/issues,
                    4.get/api/issues?sort=newest,
                    5.GET /api/issues/:id ,
                     6.PATCH /api/issues/:id, 
                     7.DELETE /api/issues/:id
